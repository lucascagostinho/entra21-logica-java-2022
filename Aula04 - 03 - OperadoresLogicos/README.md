# Aprendendo sobre operadores lógicos

## Tipos

- && Utilizado quando desejamos verificar se uma condição E outra é verdadeira ou falsa. Para resultar true, os itens da comparação, tem que dar tudo true

- || Utilizado quando desejamos verificar se uma condição OU outra é verdadeira ou falsa. Para resultar true, basta um true em uma comparação para retornar true

-> T && T = T
-> T && F = F
-> F && T = F
-> F && F = F

-> T || T = T
-> T || F = T
-> F || T = T
-> F || F = F
